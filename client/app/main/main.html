TestFlow

Front End Prototype for Spec'ing Out Manual Tests

TestFlow

A prototype I built for the VTeam at Ultimate Software, this application is designed to make spec'ing out manual tests easier. Inspired by Workflowy, it is essentially a text editor with a nested bullet structure. Users can simply add multiple steps to a test case, and move that step up and down the hierarchy with a simple tab or shift+tab. Users can also drill down into a particular case, step, or substep by clicking on its bullet.

R2-PG

HTML5 Video Game w/ AI Built on CreateJS

R2-PG

R2-PG was a class project I created with a team in AI for Computer Games during the Spring of 2013. I was responsible for the idea and a large portion of the implementation. The idea was to build a fighting game with turn-based RPG mechanics. We used n-gram AI to prevent users from using repetitive attacks. The opening screen allows you to create custom combos which cost stamina. Each move can be executed by clicking a number along the left-hand side. We used Flash to create the animations and compiled them to JavaScript/HTML5 using the CreateJS library. KnockoutJS was also used for some of the variable bindings.

Yuke

Full Stack YouTube Jukebox

Yuke

Yuke was built during an 18 hour hackathon at the University of Florida in Spring 2015. The idea was to create a 'YouTube jukebox' that would allow a group of friends to queue up songs on a central screen connected to a sound system at a party. Friends could simply use their phones to search YouTube and build a playlist. The application was built on Meteor, a full-stack JS framework with some unique isomorphic JavaScript properties. The client side UI was built on the Materialize CSS library.
